## 💻 LACOSTE SHOP 💻

### 📄 프로젝트 소개 📄
React-Router을 사용하여 디테일 페이지, 로그인 이벤트를 구현한 LACOSTE SPA사이트 입니다.
로그인 시 제품 상세페이지를 볼 수 있습니다.

### 💡 기술 스택 💡
> javascript
  - json-server : 데이터베이스 역할을 해주는 db.json파일을 생성 후 제품명, 금액 등 정보 기입하였스며, fetch를 통해 api 값을 가져오는 형태로 구현하였습니다.
  
> React
  - React-Router 
  - React-Bootstrap
  - useEffect
  - useState
